# Medication Dose Calculator 💊

A simple, fast, and responsive web-based calculator to help determine the correct dosage of Paracetamol (Progesic) and Ibuprofen (Brufen) for children based on their body weight.

## Features ✨

* **Weight-based Calculation:** Instantly calculates the safe dosage ranges (low to high) based on the child's weight in kilograms.
* **Multiple Strengths Supported:**
  * **Paracetamol:** Supports standard (120mg / 5ml) and Progesic 250 (250mg / 5ml).
  * **Ibuprofen:** Supports standard (100mg / 5ml) and Forte (200mg / 5ml).
* **Alternating Schedule Guide:** Includes a helpful 24-hour schedule demonstrating how to safely alternate between Paracetamol and Ibuprofen to maintain continuous fever control without exceeding maximum daily limits.
* **Modern Dark Mode:** Includes a sleek, iOS-inspired theme toggle switch with fluid sun/moon animations to switch between Light and Dark themes.
* **Single File Application:** Everything (HTML, CSS, and JavaScript) is bundled into a single, lightweight `progesic-calculation.html` file that can be opened in any modern browser without a server.

## How to Use 🚀

1. Open `progesic-calculation.html` in any web browser.
2. Enter the child's weight in kilograms.
3. Select the medication strengths you have on hand from the dropdown menus.
4. Click **Calculate Both** to instantly see the recommended safe dosage ranges.

## Disclaimer ⚠️

*This tool is intended for educational and reference purposes only. It is not a substitute for professional medical advice, diagnosis, or treatment. Always consult with a qualified healthcare provider or pediatrician before administering any medication to a child.*

* **Paracetamol Maximums:** Max 4 doses in 24 hours. Minimum 4 hours apart.
* **Ibuprofen Maximums:** Max 3 doses in 24 hours. Minimum 6 hours apart. Always take after food.
